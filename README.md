## Node.js lib starter

### Clone

```bash
git clone https://github.com/hawyar/node-lib-starter.git lib-name
```

### Test

```bash
npm run test
```

### Docs

```bash
npm run generate:docs
// then run npm run serv
```

### Build

```bash
npm run build
```
